### 结构

#### TCB
```
0x1A -> |----------------------|
        |TSS选择子             |
0x18 -> |----------------------|
        |                      |
        |TSS线性基地址         |
0x14 -> |----------------------|
        |TSS界限值             |
0x12 -> |----------------------|
        |LDT选择子             |
0x10 -> |----------------------|
        |                      |
        |LDT线性基地址         |
0x0C -> |----------------------|
        |LDT当前界限值         |
0x0A -> |----------------------|
        |下一个可用的线性基地址|
        |（4KB对齐）           |
0x06 -> |----------------------|
        |任务状态              |
0x04 -> |----------------------|
        |                      |
        |下一个TCB线性基地址   |
0x00 -> |----------------------|
```
